# 6.2.6 Registrando las rutas manejadas con OData

Vamos a registrar las rutas mediante las cuales van a ser accesados nuestros servicios, nuevamente en nuestro archivo **Startup** ahora en el mÃ©todo **Configure** agregamos la ruta odata para todos los modelos que registremos como EDM, permitimos tambien que los servicios permitan filtrado, order by, seleccinar solamente unos campos

{% code title="Startup.cs" %}
```csharp
public void Configure(IApplicationBuilder app, IHostingEnvironment env)
{
     app.UseMvc(b =>
            
                .Select().Expand().Filter().OrderBy().MaxTop(100).Count();
                b.MapODataServiceRoute("odata", "odata", GetEdmModel());
            });
```
